# Мыть деда

Node.js браузерная игра в вертикальном мобильном формате.

## Запуск

```bash
npm start
```

После запуска открой:

```text
http://localhost:3000
```

## Что есть

- Главный экран с ресурсами, профилем деда, настроением и меню.
- Мини-игра мойки с инструментами, чистотой, пеной и эффектами.
- Магазин с покупками за рубли.
- Настройка телевизора с качеством сигнала и деталями.
- Приготовление доширака с ингредиентами, оценкой и наградой.
- Сохранение прогресса в `localStorage`.

Фоны, комната и дед сделаны CSS-плейсхолдерами, чтобы потом можно было заменить их твоими скринами/картинками.
